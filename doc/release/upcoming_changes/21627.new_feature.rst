``casting`` and ``dtype`` keyword arguments for `numpy.stack`
-------------------------------------------------------------
The ``casting`` and ``dtype`` keyword arguments are now available for `numpy.stack`. 
To use them, write ``np.stack(..., dtype=None, casting='same_kind')``.
